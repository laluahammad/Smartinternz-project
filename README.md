# Smartinternz-project
Service now family expense management system project
ServiceNow Family Expense Management Project
Project Title
Calculating Family Expenses Using ServiceNow
Internship Platform
This project was completed as part of the internship program conducted by SmartBridge through the SmartInternz platform.
Project Description
The purpose of this project is to build a simple application in ServiceNow to manage and track family expenses.
The application allows users to record daily expenses, categorize them, and generate reports to understand spending patterns.
Features
Create a Family Expense application
Store expense details
Categorize expenses
Record amount and date
Maintain expense records
Generate reports for analysis

Added fields such as:
Expense Name
Category
Amount
Date
Paid By
Notes
Configured form layout.
Added sample expense records.
Generated reports for expense analysis.
Exported the application as an XML update set.

This project demonstrates how ServiceNow can be used to create a simple expense tracking system and manage data efficiently using custom tables and reports.
